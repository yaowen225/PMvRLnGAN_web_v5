系統介紹

PMvRLnGAN 是一個基於強化學習和圖注意力網絡的投資組合管理系統，它通過以下四個步驟來實現投資組合的優化：

  1. GAT 股票關係分析：利用圖注意力網絡分析股票之間的關聯性
  2. Stock-Picked Agent 股票選擇：選擇風險最小的股票組合
  3. TCN-AE 特徵壓縮：壓縮股票的技術指標，提取特徵
  4. Trading Agent 交易決策：基於低風險股票列表進行每日交易決策
系統特點
  • 使用預訓練模型，無需重新訓練
  • 基於風險最小化的股票選擇策略
  • 考慮股票間的關聯性
  • 提供每日交易決策建議
